Computer Graphics

Path Tracer | Spring 2020

Implemented path tracer with Monte Carlo integration in C++. Wrote code to implement BRDFs and BTDFs for diffuse, specular, and specular refractive shaders and compute radiance with next event estimation and depth of field. Terminated recursion with Russian roulette method.

Codebase provided by Shuang Zhao (https://shuangz.com/)

Cloth Simulation | Spring 2020

Implemented mass-spring particle system to simulate cloth motion in javascript using semi-implicit Euler's method. Wrote code to implement actual simulation: updating particle positions and velocities.
Forces considered: gravity, damping, viscous, structural, shear and flexion.

Codebase provided by Shuang Zhao (https://shuangz.com/)

Web Development

My Philanthropies Volunteer Web Tracker Application | Fall 2019

In a team of six, we created a web application for My-Philanthropies, a non profit organization that is a "unique network dedicated to linking philanthropreneurs to each other, creating symbiotic relationships between like-minded people who are ready to make a difference in their own lives and in our communities." The application lets volunteers create an account and view and sign up to volunteer at events.

I worked on the backend team, writing API endpoints for interactions with events such as signing up and viewing. I created the ER diagram for our database and helped write overall documentation. We used the Django REST framework and deployed our application on an AWS EC2 instance.
